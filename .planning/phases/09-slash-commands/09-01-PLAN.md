The file must have YAML frontmatter with these fields: name: comms-watch, description: "Launch comms-watch TUI in a tmux split pane (35% width, right side)", disable-model-invocation: true, allowed-tools: Bash.

Body instructions (four steps, each using the Bash tool):

**Step 1 — Check if bun is available:**
Run: `which bun`
If non-zero exit, print: `bun not found. Install Bun first: https://bun.sh`
Stop.

**Step 2 — Check if running inside tmux:**
Run: `echo $TMUX`
If empty, print: `comms-watch requires tmux. Start a session first: 'tmux new-session', then run /comms-watch again.`
Stop.

**Step 3 — Check for duplicate instance:**
Run: `tmux list-panes -s -F '#{pane_title}'`
If any line starts with `comms-watch:`, print: `comms-watch is already running in this session. Use Ctrl+C in that pane to stop it first.`
Stop.

**Step 4 — Spawn comms-watch in a new right-side pane:**
Run: `tmux split-window -h -p 35 -d "cd \"$PWD\" && printf '\\033]2;comms-watch: %s\\033\\\\' \"$PWD\" && bun tui/main.ts"`

The `printf` sets the pane title to `comms-watch: /path` so Step 3 duplicate detection works. The `-h` creates a right-side vertical split. The `-p 35` sets 35% width. The `-d` keeps focus on the original pane.

Print: `comms-watch sidebar opened.`

IMPORTANT: This is a markdown file that instructs Claude Code, NOT a shell script. Each step tells Claude what Bash command to run and how to interpret the result. Match the prose style and structure of /gsd-watch exactly.
