**Exported functions — implement ALL of these:**

1. `sendAck(myId: PeerId, orchestratorId: PeerId, taskId: number, phaseNumber: number): Promise<void>`
   - Calls `brokerFetch("/send-message", { from_id: myId, to_id: orchestratorId, text: "Received phase ${phaseNumber}, starting setup", msg_type: "status_response", payload: { task_id: taskId, status: "acknowledged", tasks_completed: 0, tasks_total: 0, current_task: "setup", last_activity: new Date().toISOString() } satisfies StatusResponsePayload })`

2. `gitPullRebase(cwd: string, branch: string): Promise<{ ok: boolean; error?: string }>`
   - Uses `Bun.spawn(["git", "pull", "--rebase", "origin", branch], { cwd, stdout: "pipe", stderr: "pipe" })`
   - Returns `{ ok: true }` on exit code 0, `{ ok: false, error: stderr }` otherwise

3. `readPlanFile(planPath: string, cwd: string): Promise<{ ok: boolean; content?: string; error?: string }>`
   - Validates `planPath.startsWith(".planning/phases/")` — if not, returns `{ ok: false, error: "Invalid plan path: must start with .planning/phases/" }`
   - Uses `Bun.file(path.join(cwd, planPath))` to read
   - Returns `{ ok: true, content }` if file exists, `{ ok: false, error: "Plan file not found: ${planPath}" }` if not

4. `checkConflicts(waveId: number, files: string[]): Promise<{ ok: boolean; conflicts?: Array<{ task_id: number; task_name: string; conflicting_files: string[] }> }>`
   - Calls `brokerFetch("/conflict-check", { wave_id: waveId, files })`
   - Returns `{ ok: true }` if conflicts array is empty, `{ ok: false, conflicts }` otherwise

5. `sendProgress(myId: PeerId, orchestratorId: PeerId, payload: PhaseProgressPayload): Promise<void>`
   - Calls `brokerFetch("/send-message", { from_id: myId, to_id: orchestratorId, text: "Task ${payload.tasks_completed}/${payload.tasks_total} complete", msg_type: "phase_progress", payload })`

6. `sendPhaseComplete(myId: PeerId, orchestratorId: PeerId, payload: PhaseCompletePayload): Promise<void>`
   - Calls `brokerFetch("/send-message", { from_id: myId, to_id: orchestratorId, text: "Phase ${payload.phase_number} complete", msg_type: "phase_complete", payload })`

7. `sendPhaseBlocked(myId: PeerId, orchestratorId: PeerId, payload: PhaseBlockedPayload): Promise<void>`
   - Calls `brokerFetch("/send-message", { from_id: myId, to_id: orchestratorId, text: "Phase ${payload.phase_number} blocked: ${payload.reason}", msg_type: "phase_blocked", payload })`

8. `sendStatusResponse(myId: PeerId, orchestratorId: PeerId, payload: StatusResponsePayload): Promise<void>`
   - Calls `brokerFetch("/send-message", { from_id: myId, to_id: orchestratorId, text: "Status: ${payload.status}", msg_type: "status_response", payload })`

9. `handleReclaim(myId: PeerId, orchestratorId: PeerId, payload: ReclaimTaskPayload, cwd: string, branch: string, tasksCompleted: number, tasksTotal: number): Promise<void>`
   - Stage all: `Bun.spawn(["git", "add", "-A"], { cwd })`
   - WIP commit: `Bun.spawn(["git", "commit", "-m", "WIP: reclaimed by orchestrator -- ${payload.reason}", "--allow-empty"], { cwd })`
   - Push with jitter: call `gitPushWithJitter(cwd, branch)` (don't care about result — best effort)
   - Get HEAD SHA: `Bun.spawn(["git", "rev-parse", "HEAD"], { cwd, stdout: "pipe" })` and read stdout
   - Send status_response with `status: "reclaimed"`, `tasks_completed`, `tasks_total`, `current_task: "reclaimed"`, `last_activity: new Date().toISOString()`

10. `gitPushWithJitter(cwd: string, branch: string): Promise<{ ok: boolean; error?: string }>`
    - Random jitter: `const jitterMs = Math.random() * 3000; await new Promise(r => setTimeout(r, jitterMs));`
    - First push attempt: `Bun.spawn(["git", "push", "origin", branch], { cwd, stdout: "pipe", stderr: "pipe" })`
    - If exit code 0: return `{ ok: true }`
    - If non-zero: rebase via `Bun.spawn(["git", "pull", "--rebase", "origin", branch], { cwd, stdout: "pipe", stderr: "pipe" })`
    - If rebase fails: return `{ ok: false, error: "Rebase failed after push rejection" }`
    - Retry push (with another jitter round): if fails return `{ ok: false, error: stderr }`
    - If succeeds: return `{ ok: true }`

11. `shouldSkipWrite(filePath: string, flags: string): boolean`
    - Returns `true` if `flags.includes("--no-transition")` AND (`filePath.endsWith("ROADMAP.md")` OR `filePath.endsWith("STATE.md")`)
    - Returns `false` otherwise

12. `callTaskStart(taskId: number, sessionId: string): Promise<void>`
    - Calls `brokerFetch("/task-start", { task_id: taskId, session_id: sessionId })`

13. `callTaskComplete(taskId: number): Promise<{ ok: boolean; wave_completed: boolean }>`
    - Calls `brokerFetch("/task-complete", { task_id: taskId })`
    - Returns the result

14. `callTaskBlocked(taskId: number, reason: string): Promise<void>`
    - Calls `brokerFetch("/task-blocked", { task_id: taskId, reason })`
