**tui/app.ts (~120 LOC):**

Create the App class that manages tab state, rendering, input, and refresh loops.

```typescript
import { clearScreen, moveTo, write, fg, bg, resetStyle, bold, C, drawHLine, getTermSize, hideCursor } from "./render.ts";
import { startInput, stopInput, type KeyEvent } from "./input.ts";

import * as peers from "./tabs/peers.ts";
import * as waves from "./tabs/waves.ts";
import * as tasks from "./tabs/tasks.ts";
import * as messages from "./tabs/messages.ts";
import * as stats from "./tabs/stats.ts";
```

**Tab definition array (order matches number keys 1-6):**
```typescript
const TABS = [gsdWatch, peers, waves, tasks, messages, stats];
```

**App class:**
- `activeTab: number` — index into TABS, default 0
- `running: boolean` — controls main loop
- `refreshTimers: Timer[]` — setInterval handles for each tab's polling
- `brokerConnected: boolean` — last known broker status
- `noEmoji: boolean` — ASCII-only mode flag

**Methods:**
- `start(noEmoji: boolean): void`
  - Store noEmoji flag
  - Call `hideCursor()`
  - Call `startInput(this.handleKey.bind(this))`
  - Set up SIGWINCH handler: `process.on("SIGWINCH", () => this.render())`
  - Start refresh timers: for each tab with REFRESH_MS > 0, setInterval that calls `this.refreshTab(i)` at that interval
  - Start a broker health check interval (every 5s): updates `this.brokerConnected`
  - Call `this.render()` for initial draw

- `stop(): void`
  - `this.running = false`
  - Clear all refresh timers
  - Call each tab's `stop()`
  - Call `stopInput()`

- `handleKey(key: KeyEvent): void`
  - If key.name === "q" or (key.name === "c" && key.ctrl): call `this.stop()` then return (main.ts handles exit)
  - If key.name is "1"-"6": switch to that tab (0-indexed: parseInt(key.name) - 1), call render()
  - If key.name === "tab": next tab (wrap around), call render()
  - If key.name === "shift-tab": previous tab (wrap around), call render()
  - If key.name === "?": toggle help overlay (stretch goal, can be deferred)
  - Otherwise: forward to active tab's handleKey(key.name)
  - Expose an `onQuit` callback property so main.ts can hook cleanup

- `render(): void`
  - Get terminal size via getTermSize()
  - clearScreen()
  - renderTabBar(): Row 1. For each tab, show `{key} {name}` with active tab highlighted (purple underline effect: bg(C.bgLight) + fg(C.bright) for active, fg(C.gray) for inactive). Draw separator line on row 2.
  - Render active tab content: call TABS[activeTab].render(3, 1, cols, rows - 3) — content area starts row 3, ends row (rows - 1) to leave room for status bar
  - renderStatusBar(): Last row. Left side: broker status (green "BROKER OK" if connected, red "BROKER --" if not) + BROKER_URL. Right side: "q:quit  ?:help  1-6:tabs"

- `refreshTab(index: number): void`
  - If index === activeTab, call render() (only re-render if the refreshing tab is visible)
