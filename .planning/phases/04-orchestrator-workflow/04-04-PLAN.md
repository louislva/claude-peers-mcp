4. "groups independent phases into same wave" — Create 3 PhaseNodes with no deps. Assert: 1 wave containing all 3.
5. "chains dependent phases into sequential waves" — Phase A (no deps), Phase B (depends on A), Phase C (depends on B). Assert: 3 waves, one phase each.
6. "parallelizes phases with different dependencies" — Phase A, Phase B (no deps), Phase C (depends on A), Phase D (depends on B). Assert: Wave 1 = [A, B], Wave 2 = [C, D].

8. "filters out completed phases" — 5 phases, 3 completed. Assert: only pending phases appear in waves; completed deps don't create edges.

10. "splits conflicting phases into sub-waves" — Phase A modifies `package.json`, Phase B also modifies `package.json`. Assert: returns 2 sub-waves, one phase each.
11. "handles three-way conflict correctly" — A conflicts with B, B conflicts with C, A does not conflict with C. Assert: A and C in same sub-wave, B in separate sub-wave.
